Selectores de clase

A partir de esta página no usaremos más el atributo style en ninguna etiqueta.

Todo el estilo aplicado en el documento será definido dentro de una etiqueta style como "clase", anidada en la etiqueta head del documento.

p { /*El default para cualquier párrafo*/
    font-size: 1em;
    margin-bottom:60px;
}

Todos los párrafos tienen un default como estilo que incluyen un margin-bottom para separarlos más entre ellos.

.claseResaltado {
    font-size:1.2em;
    font-weight:bold;
    background-color: yellow;
}

Este párrafo toma el estilo definido en la etiqueta style en el header del documento como class="claseCita"

Este párrafo toma el estilo definido en la etiqueta style en el header del documento como class="claseResaltado"

Este párrafo toma el estilo definido en la etiqueta style en el header del documento como class="claseAyuda"

Este párrafo toma el estilo definido en la etiqueta style en el header del documento como class="claseContenidoComun"

Párrafo formateado por un estilo particular definido en el style del documento para un párrafo específico id="idUnico". No se trata de un selector de clase sino de un selector de elemento "#"
